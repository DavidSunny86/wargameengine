
```
void EnableMSAA()
```

## Summary ##
Enables the fullscreen [MultiSample Anti-Aliasing](http://en.wikipedia.org/wiki/Multisample_anti-aliasing).

## Parameters ##
NONE

## Output values ##
NONE

## Errors ##
  * **no arguments expected**. Parameters number or type does not match.

## Example ##
```
EnableMSAA()
```
Enables the MSAA.
