
```
void DisableMSAA()
```

## Summary ##
Disables the fullscreen MSAA. It can be enabled with a [DisableMSAA](LuaDisableMSAA.md) function.

## Parameters ##
NONE

## Output values ##
NONE

## Errors ##
  * **no arguments expected**. Parameters number or type does not match.

## Example ##

```
EnableMSAA()
...
DisableMSAA()
```
Disable MSAA.
